# FAQ

## Preciso de cloud para o sistema funcionar?
Não. O projeto foi desenhado para processamento e armazenamento local.

## Posso começar sem Kubernetes?
Sim. Comece com Docker Compose (`src/docker-compose.yml`) e evolua para K3s quando necessário.

## Quais protocolos principais o sistema usa?
MQTT, Zigbee, RTSP e ONVIF.

## O módulo de drones é obrigatório?
Não. É uma camada avançada e opcional.

## Como contribuir?
Abra issues/PRs e siga as diretrizes do repositório.
