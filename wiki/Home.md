# Home Security DIY Wiki

Bem-vindo à wiki do **Home Security DIY** — sistema residencial open source com processamento 100% local.

## Visão geral

O projeto combina automação, vídeo inteligente e sensores para segurança em camadas:

- **Home Assistant** para automações e dashboard.
- **Frigate** para NVR e detecção por IA local.
- **Zigbee2MQTT** para sensores sem fio.
- **Mosquitto** para mensageria MQTT.
- **Docker/K3s** para execução em dev/prod.

- Cenários residenciais: **Rural, Urbano e Apartamento**.
- Defesa em profundidade com camadas passiva, ativa e reativa.
- Privacidade por design: sem cloud obrigatório.
