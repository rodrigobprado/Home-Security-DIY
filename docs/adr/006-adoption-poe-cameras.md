# 6. Adoção de Câmeras PoE em vez de Wi-Fi

Data: 2026-02-22

## Status

Aceito

## Contexto

Confiabilidade e resistência a interferência são críticas para segurança residencial.

## Decisão

Priorizar câmeras PoE (Ethernet) em produção.

## Consequências

### Positivas
- Menor suscetibilidade a jamming RF.
- Estabilidade de link superior.
- Alimentação e dados no mesmo cabo.

### Negativas
- Exige cabeamento estruturado.
- Custo inicial maior que câmeras Wi-Fi básicas.

## Mitigação
- Projeto de rede com switch PoE dedicado.
- Uso de VLAN isolada para câmeras.
